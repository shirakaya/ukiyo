# 《时政通讯》游玩手册

> \*"此身、此心，本就是为了如您所愿而存在的。"\*

\---

## 一、游戏简介

《时政通讯》是一款基于《刀剑乱舞》世界观创作的**单页交互式视觉小说**。你将以审神者的视角，通过一台时空紊乱的终端设备，与代号 **J-0108** 的检察官（数珠丸恒次）以及加州清光进行通讯，逐步揭开本丸深处被遗忘的真相。

**核心体验**：在手机界面中探索、通讯、跳转时间，收集散落的文件与记忆，最终导向不同的命运终点。

## 三、核心机制

### 1\. 时间跳转（最重要）

点击日历右上角的 **⚙️（设置按钮）** → 输入年/月/日 → **跳转**。这是推动剧情的核心手段。

**可跳转范围**：公元 2000 年 \~ 2250 年。低于 2000 年会提示「终端损毁」。

**时间跳转的影响**：

* 每次跳转后，地图坐标会随机变化
* 跳转次数过多（≥9次）屏幕会出现故障特效（glitch）
* 跳转 ≥20 次且未授权定位 → 可能触发 **「迷途」** 结局
* 特定日期会触发新邮件、解锁新文件、开启新镜头

### 2\. 文件解锁与日期

许多文件只有在特定日期范围内才会出现在「文件」应用中：

|文件|解锁日期范围|内容|
|-|-|-|
|关于那一天.txt|2024-01-19 \~ 2026-11-03|关于记忆与伤势的独白|
|健康诊断书.pdf|2213-07-13 \~ 2214-01-18|诊断报告图片|
|独白.mp3|任意歌曲播放后|日期=2214-01-18显示正确文本，否则乱码|
|档案 NSC-2225-REP-0108|≥ 2225-07-03|检察官档案图片|

### 3\. 电量系统

电量会随时间自动下降（每30秒固定减1），**无法充电**。当电量降至 0% 时，强制触发结局 **「如是观」**（Reload End）。

> ⚠️ \*\*提示\*\*：电量是真实的时间限制。如果希望充分探索，请留意剩余电量，避免不必要的等待。

## 四、系统详解

### 📧 邮件系统

* **收信**：部分邮件会自动送达（如时间跳转后、首次交互后）
* **发信**：点击「写邮件」→ 填写收件人和内容 → 发送

**关键操作**：

* 收件人填写 **J-0108** 并发送 → **解锁 J-0108 聊天对象**
* 内容包含「救救我 / help / sos」→ 标记求救状态（影响结局）
* 收件人含作者关键词 + 内容含 tip → 收到「终端使用须知」彩蛋邮件

**隐藏角色：笑面青江**
解锁 J-0108 后，向收件人 **「笑面青江」** 发送任意文本，可在特定日期触发一系列邮件。这些邮件以中日双语书写，讲述了数珠丸恒次从察觉、隐忍到离别本丸的完整暗线故事。邮件共 16 封，按时间顺序逐步解锁，是理解全剧情的重要补充。

|邮件主题|触发日期|内容概要|

### 💬 聊天系统（J-0108）

与 J-0108 的聊天是剧情核心。

**首次进入**：J-0108 会发送引导消息，并弹出 **「是否授权定位」** 按钮：

* **同意**：发送地图截图，60秒后他会说「未发现您的踪迹...我们真的在同一时空吗？」
* **拒绝**：他会表示理解，并在通讯中尽量指引你

**关键词回复**：输入特定词汇可触发 J-0108 的专属回应。已收录 45 组关键词，例如：

* 「你好 / 在吗」 → 问候回应
* 「累 / 困 / 疲惫」 → 关心慰问
* 「帮帮我 / 救救我」 → 「我一定会找到您」
* 「记忆 / 忘记 / 回忆」 → 关于记忆脆弱性的独白
* **「我喜欢你 / 我爱你 / 想见你」** → 可能导向 **垂慈摄受** 结局（需满足条件）

**发送附件**：点击输入框左侧的 **+** 按钮，可发送相册中的照片或文件：

* 每张照片/文件都会触发 J-0108 的独特反应，推动剧情
* 部分附件反应是解锁结局的必要条件

### 💬 聊天系统（加州清光）

* **解锁**：在日历中选择日期 **2205-01-14**
* 无关键词系统，无特殊结局，作为支线角色存在

### 📅 日历系统

* **点击日期格子**：切换当前日期，更新时钟显示
* **添加备注**：在底部输入文字 → 点击 + 添加
* **删除备注**：点击备注右侧的 ✕（部分特殊备注不可删除）

**特殊日期**：

* **2026-11-03**：默认存在「前往本兴寺虫干会」备注，删除后触发虫干会彩蛋
* **任意年-04-13**：添加备注会自动追加「恒次文化财指定日」彩蛋
* **2023-01-05**：时间跳转至此日期，收到开发者「客串事件」邮件

### 📁 文件系统

点击文件条目可展开/收起内容。阅读文件会设置「已读」标志，部分结局需要「已读」但「未触发聊天反应」的 paradox 状态。

### ◐ 相机系统

* **基础镜头**：天守阁附近、自拍
* **解锁镜头**：

  * 拍摄「天守阁附近」后，追加「大広间」镜头
  * 拍摄「大広间」后，追加「御神木」镜头
* **限时镜头**：

  * 日期 2024-04-13 \~ 2026-11-03：追加「莲花」镜头
  * J-0108 已解锁 且 日期 = 2026-11-03：追加「本兴寺」镜头（**无法保存**）

### ✧ 地图系统

* 支持拖拽平移、滚轮/双指缩放
* 定位授权后，地图标记会同步变化

## 五、结局攻略（全8种）

### ⚡ 惩罚结局

|结局|类型|触发方式|说明|
|-|-|-|-|
|**如是观**|Reload End|电量降至 0%|终端关机，一切归零。最快速的"通关"方式，也是惩罚。|
|**迷途**|Sad End|时间跳转 ≥ 20 次 且 **未授权定位**|在时空中迷失，忘记为何启航，一如过往。|
|**徒然无所得**|Normal End|发送 SOS 后，在设置中**重启 3 次**|成功获救回到自己的时代，但没能见到那位检察官。|

### 🌸 剧情结局

#### 空花阳焰（Open End）

> \*"说出那句话的时候，有什么东西在我深处裂开了。"\*

* **输入**：「我该走了 / 离开 / 回家」
* **条件**：

  * 已读过所有文件（日记、诊断书、独白、档案）
  * **但从未在聊天中发送过这些文件**（paradox 状态）
  * 未触发任何附件反应（!diary \&\& !healthReport \&\& !monologueCorrect \&\& !archive）
  * 未授权定位、未识破真名
  * **已发送 SOS**
* **攻略**：先读文件，然后发邮件求救，再输入离开。

#### 须弥芥子（Bad End）

> \*"真正的悲剧并非身处地狱，而是在极乐门前知晓了地狱的真相。"\*

* **触发**：满足特定条件后，输入**任意普通文本**即可触发
* **条件**：

  * 核心文件全收集（diary + healthReport + archive）
  * 已发送乱码独白，**未发送正确独白**（monologueGarbled \&\& !monologueCorrect）
  * 后期条件达成（识破真名 + 授权定位 + 发送自拍反应）
  * 未发送 SOS
  * **未阅读失踪检察官邮件**
* **注意**：此结局与「一莲托生」部分条件重叠，但不需要收集正确独白，也无需触发虫干会彩蛋和阅读失踪邮件。若同时满足一莲托生的全部条件（包括正确独白、虫干会彩蛋、已读失踪邮件），则优先进入一莲托生。

#### 垂慈摄受（Good End）

> \*"那简直就像是真言，是将我这个存在，束缚于此世，并赋予我意义的，独一无二的真言。"\*

* **输入**：「我喜欢你 / 我爱你 / 想见你」
* **条件**：

  * 已发送日记文件反应（diary）
  * 已发送档案反应（archive）
  * 已识破真名（keywordRevealed）
  * 已授权定位（locationAgreed）
  * **未发送 SOS**
* **攻略**：与 J-0108 聊天时发送日记和档案，跳转时间触发真名识破，同意定位，然后输入告白。

#### 酬愿（Happy End）

> \*"无论你是不是数珠丸恒次，无论我是不是审神者，我们魂魄的某处，或许也会被遥远过去的因果所牵引。"\*

* **输入**：「我回来了」
* **条件**：

  * 核心文件全收集 + 双独白收集（正确独白 + 乱码独白）+ 后期条件（识破真名 + 授权定位 + 自拍反应）
  * **未发送 SOS**
  * **未触发虫干会彩蛋**（easterEggTriggered = false）
  * **当前日期 ≥ 2216年11月3日**
* **攻略**：完成全收集，确保没有删除虫干会备注触发彩蛋，将日期跳转至 **2216年11月3日或之后**，输入「我回来了」。

#### 一莲托生（True End）

> \*"若您的灵魂不寻求救赎，我便与您一同堕入地狱。那便是，我们一莲托生之契约。"\*

* **触发**：满足全部条件后，输入**任意文本**即可触发（优先级最高）
* **条件**：

  * 核心文件全收集（diary + healthReport + archive）
  * 双独白收集（monologueGarbled + monologueCorrect）
  * 后期条件（keywordRevealed + locationAgreed + 自拍反应 bg3）
  * **未发送 SOS**
  * **已触发虫干会彩蛋**（easterEggTriggered = true）
  * **已阅读失踪检察官邮件**（missingInspectorMailRead = true）
  * **收到笑面青江的回复邮件**（emaillReplyReceived = true）
* **攻略**：

  1. 解锁并阅读所有文件，在聊天中发送触发全部附件反应
  2. 跳转时间 ≥5 次后，输入「数珠丸恒次」识破真名
  3. 同意定位授权，发送自拍照片触发 bg3 反应
  4. 删除日历中 2026-11-03 的「前往本兴寺虫干会」备注，触发虫干会彩蛋
  5. 阅读失踪检察官邮件（日期 ≥ 2216-11-14 自动收到）
  6. 向**笑面青江**发送邮件，收到至少一封回复（如「关于那个人的事」）
  7. 确保从未发送 SOS
  8. 输入任意消息，触发 True End

## 六、隐藏内容 \& 彩蛋

### 1\. 真名识破

在与 J-0108 聊天时，输入包含「数珠丸恒次 / 数珠丸 / 恒次 / juzumaru」等词汇，且满足：

* 已阅读档案（archiveRead）
* 时间跳转 ≥ 5 次

他会回应：「被您看穿了...宛如宿命一般」，并解锁 keywordRevealed 标志。

### 2\. 虫干会彩蛋

在日历中找到 **2026-11-03** 的「前往本兴寺虫干会」备注，**删除它**。会触发一系列不可删除的红色备注，自动标记 easterEggTriggered。

### 3\. 文化财指定日彩蛋

在日历中添加备注，日期选择 **任意年的 4月13日**，会自动追加「今天可是恒次的重要文化财指定日！」粉色备注。

### 4\. 开发者邮件

时间跳转至 **2023-01-05**，会收到一封来自开发者的碎碎念邮件「客串事件」。

### 5\. 笑面青江邮件系列（暗线剧情）

解锁 J-0108 后，向 **「笑面青江」** 发送邮件，可触发一条完整的暗线叙事。这 16 封邮件以中日双语呈现，从 2213 年 5 月延续至 2216 年 1 月，完整记录了数珠丸恒次从**察觉主人灵力溃散**、**隐忍不言**、**调任离别**到\*\*最终在新本丸被当作「传说中的刀」\*\*的全过程。

**核心叙事节点**：

* **2213年夏秋**：数珠丸开始察觉异常，在佛间彻夜祈愿，在主人房前驻足却不敢打扰
* **2214年1月初**：调任通知下达，1月8日离开本丸。离别时只说了一句「请多保重」
* **2214年4月后**：在新本丸，没人知道他曾经待过那所本丸
* **2215年1月**：那封「不会寄出的信」——「他一次都没有恨过你」
* **2216年1月**：笑面青江终于回答了你的疑问——「决定寄出去，是因为这封信到了该到的时候」

> 这条暗线是理解数珠丸恒次这个角色、以及本丸「被隐藏的历史」的关键拼图。同时，\*\*收到笑面青江的回复邮件\*\*也是触发「一莲托生」（True End）的必要条件之一。邮件内容中日双语分行显示，日语以淡紫色呈现，中文以深紫色呈现，便于阅读区分。

### 6\. 故障特效

随着时间跳转次数增加，屏幕会逐渐出现 glitch 干扰层：

* ≥9次：轻微干扰（10%透明度）
* ≥17次：中度干扰（20%透明度）
* ≥25次：严重干扰（30%透明度 + 闪烁）

## 七、新手建议流程

1. **开局**：等待 25 秒，收到第一封邮件。点击任意应用触发「拂晓的应答」邮件。
2. **解锁 J-0108**：进入邮件 → 写邮件 → 收件人填 **J-0108** → 发送任意内容。
3. **进入聊天**：点击时政通讯 → 与 J-0108 对话 → 选择**授权定位**（推荐，方便推进剧情）。
4. **解锁笑面青江邮件（可选暗线）**：解锁 J-0108 后，进入邮件 → 写邮件 → 收件人填 **笑面青江** → 发送任意内容。根据当前日期，逐步解锁 16 封暗线邮件。建议按以下顺序跳转：

   * 2213-05-05 → 2213-07-16 → 2213-08-13 → 2213-09-02 → 2213-10-03 → 2213-11-15
   * 2214-01-01 → 2214-01-03 → 2214-01-05 → 2214-01-08 → 2214-01-14 → 2214-01-19
   * 2214-04-11 → 2215-01-18 → 2216-01-13
5. **跳转时间**：进入日历 → 设置按钮 → 跳转到 **2024-01-19** 之后，解锁日记文件。
6. **探索文件**：跳转到不同日期范围，解锁健康诊断书、档案等。
7. **播放音乐**：在主屏点击播放歌曲，解锁独白文件。
8. **发送附件**：在聊天中发送解锁的照片和文件，触发 J-0108 的反应。
9. **收集反应**：确保 diary、healthReport、archive、monologueCorrect、bg3（自拍）都有触发记录。
10. **识破真名**：跳转 ≥5 次后，输入「数珠丸恒次」。
